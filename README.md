Blogicum

Blogicum — веб-приложение для публикации и управления пользовательским контентом, разработанное на Django.

Проект представляет собой многофункциональную блог-платформу, позволяющую пользователям создавать публикации, взаимодействовать с другими авторами через комментарии, управлять собственным профилем и публиковать записи по различным категориям.

Проект был разработан для изучения возможностей Django Framework, работы с базой данных через ORM, организации пользовательских взаимодействий и реализации полноценного веб-приложения по архитектуре MVT (Model–View–Template).

Функциональность
1. Работа с публикациями
2. создание публикаций;
3. редактирование собственных публикаций;
4. удаление публикаций;
5. добавление изображений к записям;
6. просмотр детальной информации о публикации;
7. отложенная публикация записей по дате;
8. управление видимостью публикаций.
9. Работа с комментариями
10. создание комментариев;
11. редактирование собственных комментариев;
12. удаление комментариев;
13. отображение комментариев на странице публикации.

Пользователи
1. регистрация новых пользователей;
2. авторизация и выход из системы;
3. просмотр профилей пользователей;
4. редактирование собственного профиля;
5. отображение публикаций автора.

Категории
1. распределение публикаций по категориям;
2. просмотр публикаций конкретной категории;
3. скрытие неопубликованных категорий.

Дополнительные возможности
1. пагинация публикаций;
2. обработка пользовательских ошибок;
3. административная панель Django;
4. система управления контентом через Django Admin.

Запуск проекта
Клонировать репозиторий:
git clone https://github.com/MikhailStand/Blogicum_final.git

Перейти в директорию проекта:
cd Blogicum_final-main

Создать виртуальное окружение:
python -m venv venv

Активировать виртуальное окружение.
Для Linux/macOS:
source venv/bin/activate

Для Windows:
venv\Scripts\activate

Установить зависимости:
pip install -r requirements.txt

Перейти в директорию проекта:
cd blogicum

Применить миграции:
python manage.py migrate

Создать суперпользователя:
python manage.py createsuperuser (user, user@gmail.com, user1234)

Запустить сервер разработки:
python manage.py runserver

После запуска проект будет доступен по адресу:
http://127.0.0.1:8000/

Административная панель:
http://127.0.0.1:8000/admin/
